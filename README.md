# Life OS

A personal operating system for organizing goals, projects, knowledge, habits, and decisions — run through Claude Code.

## Quick Start

Open this folder in Claude Code and just talk. Capture thoughts, plan your day, review your week. Claude keeps everything organized automatically.

## How It Works

- **Capture** anything → it lands in `inbox/`
- **Organize** happens automatically → items flow to Projects, Goals, Knowledge, Journal, Resources
- **Review** on a cadence → daily, weekly, monthly, quarterly, yearly
- **Never lose anything** → outdated items get archived, not deleted

## Commands

`/capture` `/today` `/week` `/review` `/goals` `/projects` `/focus` `/brain` `/cleanup` `/status` `/plan` `/reflect` `/prioritize`

See [CLAUDE.md](./CLAUDE.md) for the full configuration and operating rules.

## Structure

```
inbox/  daily/  weekly/  monthly/  quarterly/  yearly/
goals/  projects/  areas/  knowledge/  resources/
decisions/  journal/  habits/  systems/  templates/
reviews/  archive/
```

## Daily Loop

1. `/today` — generate your plan
2. Work, capturing thoughts as they come
3. `/reflect` — close the day with a journal entry

## Weekly Loop

1. `/review` (weekly) — look back
2. `/week` — plan ahead
