# Reviews

> Cadenced reflection. Each review: Achievements · Missed Goals · Lessons · Metrics · Recommendations · Focus for Next Period.
> Use [templates/review.md](../templates/review.md).

## Cadence
| Type | When | File pattern |
|------|------|--------------|
| Daily | End of day | `daily-YYYY-MM-DD.md` |
| Weekly | Sunday | `weekly-YYYY-Www.md` |
| Monthly | Last day | `monthly-YYYY-MM.md` |
| Quarterly | End of quarter | `quarterly-YYYY-Qn.md` |
| Yearly | Dec 31 | `yearly-YYYY.md` |
